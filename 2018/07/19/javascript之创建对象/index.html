javascript之创建对象

发表于 2018-07-19 | 分类于 学习笔记

工厂模式

缺点：对象无法识别（即怎样知道一个对象的类型），因为所有的实例都指向一个原型

构造函数模式

与工厂模式的不同点

1.没有显示的创建对象；
2.直接将属性和方法赋给了this对象；
3.没有return语句；

优点：实例可以识别为一个特定的类型
缺点：每次创建实例时，每个方法都要被创建一次 this.sayName = new Function()

构造函数模式优化

优点：解决了每个方法都要被重新创建的问题
缺点：如果对象定义很多方法，就要定义很多个全局函数。这样没有封装性可言……
